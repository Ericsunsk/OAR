## 目录规则

| 目录 | 内容 |
| --- | --- |
| [`product/`](product/) | 产品定位、PRD、体验、市场判断和用户工作流 |
| [`architecture/`](architecture/) | 技术架构、安全权限、记忆架构和跨模块工程决策 |
| [`validation/`](validation/) | 阶段计划、验证报告、实验结论和通过标准 |
| [`reference/`](reference/) | 外部资料、竞品、飞书/Lark、A2A 等参考来源 |

新增文档时优先放入上述目录；只有跨全项目的入口说明才放在 `docs/` 根目录。
